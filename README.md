# Habit Tap

Habit Tapは、毎日の小さな習慣をかんたんに記録するためのWebアプリです。

「水を飲む」「勉強する」「運動する」「薬を飲む」など、続けたい習慣を登録して、今日できたものをワンタップでチェックできます。

## アプリの特徴

- ログインなしですぐ使える
- 習慣を追加、編集、削除できる
- 今日の達成状況を一覧で確認できる
- 達成数、未達成数、達成率が見える
- 習慣ごとの連続達成日数がわかる
- リマインド時刻を設定できる
- ブラウザ通知で習慣の実行を思い出せる
- データはブラウザ内に保存される
- スマホでもPCでも使いやすい

## 想定ユーザー

- 勉強や運動を続けたい人
- 健康管理をしたい人
- 薬や水分補給などを忘れたくない人
- 毎日の小さな目標を見える化したい人

## 通知について

通知を使うには、ブラウザで通知を許可する必要があります。

このMVPでは、アプリを開いている間に設定時刻を確認し、未達成の習慣があれば通知します。通知が使えない環境では、アプリ内で未達成の習慣を確認できます。

## データ保存について

登録した習慣や達成履歴は、利用しているブラウザの`localStorage`に保存されます。サーバー送信やクラウド同期は行いません。
