# Habit Tap

Habit Tapは、毎日の小さな習慣をかんたんに記録するためのWebアプリです。

「水を飲む」「勉強する」「運動する」「薬を飲む」など、続けたい習慣を登録して、今日できたものをワンタップでチェックできます。

## 主な機能

- 習慣の追加、編集、削除
- 今日の達成、未達成、スキップの記録
- 今日の達成数、スキップ数、未達成数、達成率の表示
- 過去7日分の週間カレンダー表示
- 習慣の上下並び替え
- 習慣ごとの今日のメモ
- 習慣ごとの連続達成日数
- リマインド時刻の設定
- ブラウザ通知
- ブラウザ内保存
- スマホ、PC対応レイアウト

## スキップ機能

体調不良や予定変更などで実行しない日を「スキップ」として記録できます。

スキップ日は連続記録を止めない扱いにしているため、無理なく習慣を続けたい人に向いています。

## メモ機能

習慣ごとに、その日の短いメモを残せます。

例:

- 今日は15分できた
- 外出中なので軽めに実施
- 明日は朝にやる

## 通知について

通知を使うには、ブラウザで通知を許可する必要があります。

このMVPでは、アプリを開いている間に設定時刻を確認し、未達成の習慣があれば通知します。

## データ保存について

登録した習慣、達成履歴、スキップ、メモは、利用しているブラウザの`localStorage`に保存されます。サーバー送信やクラウド同期は行いません。
